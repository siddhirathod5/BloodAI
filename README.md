# BloodAI

## Overview

An intelligent Blood Bank Management System that manages donors, blood inventory, blood requests, donor eligibility, emergency donor recommendation, and blood demand prediction.

## Features

- Donor Registration
- Donor Eligibility Check
- Blood Inventory Management
- Blood Request Management
- Emergency Donor Finder
- Blood Demand Prediction
- Donation History Tracking
- Blood Usage Tracking
- FastAPI Backend
- MySQL Database

## Tech Stack

### Backend
